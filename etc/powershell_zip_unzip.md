## how to zip and unzip files using powershell

### zip
``` Compress-Archive -Path <PathToFiles> -DestinationPath <PathToDestination> ```

### unzip
``` Expand-Archive -Path <PathToZipFile> -DestinationPath <PathToDestination> ```
