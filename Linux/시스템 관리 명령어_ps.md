ps
===
프로세스 정보 표시  
  
## 주요 옵션  
-e : 현재 실행중인 모든 프로세스 표시  
-f : 유저명, 시작시간 표시  
-u : 유저명으로 검색  
-o : 사용자가 설정한 포맷으로 출력  
-p[pid] : 지정한 프로세스 정보만 표시  
-e f : 프로세스의 관계를 트리형식으로 표시  
  
## 사용예제  
### 현재 실행중인 모든 프로세스 표시  
```
$ ps -e
```

### 특정 프로세스 표시  
```
$ ps -p
```

### 유저명 시작시간 표시  
```
$ ps -f
```

### 프로세스 관계 트리 표시  
- ```-```를 이용한 옵션과 별도로 ```f```를 이용해 처리  
- ```-e```옵션으로 모든 프로세스가 표현되고, ```f```로 트리 관계 표현됨  

```
$ ps -e f
```

### 사용자 설정 포맷  
- ```-o```옵션 이용해 현재 실행중인 프로세스의 정보 중 원하는 것만 확인 가능  

user : 유저명  
pid	: 프로세스 ID  
ppid : 부모 프로세스 ID  
rss	: 실제 메모리 사용량  
pmem : 메모리 사용률  
pcpu : CPU 사용률  
time : CPU 사용 시간  
etime : 총 실행 시간  
cmd	: 커맨드
